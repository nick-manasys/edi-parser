USAGE OF THE DISKETTE

- The file AUDIT-D.95B contains the Directory Audit Team Statement for this
directory. 

- The file CONTENTD.95B is the table of contents of the UN/EDIFACT Draft
Directory.


- The files PART2_D.ZIP, PART3_D.ZIP, PART4_D.ZIP, TRMD.ZIP, 
TRSD.ZIP, TRCD.ZIP, TRED.ZIP, UNCL.ZIP and UNSL.ZIP are compressed 
versions in ASCII (code page 437 United States) of the UN Trade Data
Interchange Directory using the compression program PKZIP. The contents of
these files is described in the file CONTENTD.95B. 

To get the decompressed version of the Directory, use the program PKUNZIP.EXE 
stored on the diskette.

Type the commands:

 PKUNZIP [x:]PART2_D.ZIP  [y:][path]
 PKUNZIP [x:]PART3_D.ZIP  [y:][path]
 PKUNZIP [x:]PART4_D.ZIP  [y:][path]
 PKUNZIP [x:]TRMD.ZIP     [y:][path]
 PKUNZIP [x:]TRSD.ZIP     [y:][path]
 PKUNZIP [x:]TRCD.ZIP     [y:][path]
 PKUNZIP [x:]TRED.ZIP     [y:][path]
 PKUNZIP [x:]UNCL.ZIP     [y:][path]
 PKUNZIP [x:]UNSL.ZIP     [y:][path]
  
 where
  x:      is the drive on which the diskette is stored
  y:      is the drive on which the decompressed version will be stored.
  path    specifies the path to the directory where the decompressed
          version will be stored.
