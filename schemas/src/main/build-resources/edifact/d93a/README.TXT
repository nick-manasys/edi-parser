                      UN/EDIFACT DRAFT DIRECTORY

USAGE OF THE DISKETTE                                

The file CONTENTD.93A is the table of contents of the UN/EDIFACT Draft
Directory.

The following files are cross reference tables and do not belong to
the Draft layout of the UN/EDIFACT Directories:

  TRSDX.93A Cross reference table of use of segments within messages.
  TRCDX.93A Cross reference table of use of composite data elements
            within segments.
  TREDX.93A Cross reference table of use of data elements within
            composite and segments.

The files PART1_D.ZIP, PART2_D.ZIP, PART3_D.ZIP, PART4_D.ZIP, TRMD.ZIP, 
TRFR.ZIP, TRSD.ZIP, TRCD.ZIP, TRED.ZIP, UNCL.ZIP, DIRDEF_D.ZIP are
compressed versions in ASCII (code page 437 United States) of
the UN Trade Data Interchange Directory using the compression program PKZIP.
The contents of these files is described in the file CONTENTD.93A. To get
the decompressed version of the Directory, use the program PKUNZIP.EXE
stored on the diskette.

Type the commands:

 PKUNZIP [x:]PART1_D.ZIP  [y:][path] 
 PKUNZIP [x:]PART2_D.ZIP  [y:][path]
 PKUNZIP [x:]PART3_D.ZIP  [y:][path]
 PKUNZIP [x:]PART4_D.ZIP  [y:][path]
 PKUNZIP [x:]TRMD.ZIP     [y:][path]
 PKUNZIP [x:]TRFR.ZIP     [y:][path]
 PKUNZIP [x:]TRSD.ZIP     [y:][path]
 PKUNZIP [x:]TRCD.ZIP     [y:][path]
 PKUNZIP [x:]TRED.ZIP     [y:][path]
 PKUNZIP [x:]UNCL.ZIP     [y:][path]
 PKUNZIP [x:]DIRDEF_D.ZIP [y:][path] 
 
 where
  x:      is the drive on which the diskette is stored
  y:      is the drive on which the decompressed version will be stored.
  path    specifies the path to the directory where the decompressed
          version will be stored.
